principais formatações

Negrito

para deixar algo em negrito ou em destaque utilizamos a tag:
<strong>

Itálico

para deixar algo em itálico utilizamos a tag:
<em >

texto marcado

para marcarmos um texto utilizamos a tag <mark>

texto grande e pequeno

para um texto ficar pequeno utilizamos a tag <small>

para um texto ficar grande usamos a tag <big>

texto riscado

para deixar um texto riscado usamos a tag <del>

texto sublinhado

para sublinhar um texto usamos a tag <ins>

texto sobrescrito

para deixar um texto sobrescrito usamos a tag <sup>
ex: x20

para deixar um texto subscrito usamos a tag <sub>
ex: h2o

todas essas tags devem ser usadas de acordo com o valor semântico
