Outras formatações

código fonte

para inserir um código de maneira que todas as letras fiquem da mesma largura, utilizamos a tag <code> ex:
val=int(input('digite um número')

se quisermos deixar ele mais organizado ainda usamos a tag <pre> ex: 


    val=int(input('digite um número')
    print('o número que você escolheu foi o {}'.format(val))

citações

para fazer uma citação usamos a tag <q> ex:
como diz um amigo: o computador é um burro muito rápido

citações completas

para fazer uma citação completa usamos a tag<blockquote> ex:

html não é uma linguagem de programação, e sim uma maneira de desenvolver um site.

abreviações

para dizer o que significa uma abreviação usamos a tag <abbr> ex:
NY BR
