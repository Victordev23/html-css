Tipos de listas

listas ordenadas

para colocar uma lista ordenada usamos a tag <ol>
para mudar o estilo da lista ordenada usamos o comando type="" dentro da tag ol

os estilos disponiveis na lista ordenada são:

  1. A
    • ordena a lista em ordem alfabetica com letra maiuscula
  2. a
    • ordena em ordem alfabetica em letras minusculas
  3. I
    • ordena a lista em ordem de números romanos maiusculos
  4. i
    • ordena a lista em ordem de números romanos minusculos
  5. 1
    • ordena a lista em ordem númeral

    para começar um lista em uma ordem diferente usamos o comando start dentro da tag ol ou ul e colocamos em qual posição numerica ele deve começar

    ex:

    1. aqui coloquei para iniciar no G
    2. aqui h
    3. aqui i
    4. aqui j
    5. etc...

para inserir um objeto na lista usamos a tag <li>

listas não ordenadas

para colocar uma lista não ordenada usamos a tag <ul>

a tag ul conta com os seguintes estilos:


  • disc
  • circle
  • square

listas de definição

para criar uma lista de definição usamos a tag <dl>
para criar um termo usamos a tag<dt>
