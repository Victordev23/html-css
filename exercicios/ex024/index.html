Teste com iframe

A tag do Iframe é:
<iframe> </iframe>

Os iframes tem alguns atributos como src, height, width, name, frameborder, etc.

src e frameborder

No src podemos adicionar links(Os que são aceitos), páginas html locais, documentos em PDF, videos, etc.

Já o frameborder define a "borda" do iframe, sendo:
frameborder="0" não irá aparecer nenhuma borda, somente se definimos no css.
frameborder="1" qualquer valor diferente de zero sendo positivo vai criar uma "borda" de 1 px em volta do iframe. Ex
